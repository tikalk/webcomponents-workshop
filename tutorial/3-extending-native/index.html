Tutorial 3: Extending Native Elements

Add custom behavior to existing native HTML elements

This example shows how to create a customized element - a custom element that extends native HTML element.
Below, there is a SPAN element that is extended by a custom element, listening to key-up events, adding a word-count tag
After the SPAN element, there is an extended INPUT element, showing dollar sign for every input and removing any non-digit character

<span is="word-count">
Hello, I'm an extended SPAN element! I am editable, click me and start typing. 

            class WordCount extends HTMLSpanElement { ... };
            customElements.define('word-count', WordCount, { extends: 'span' });
<input is="dollar-input"> 

            class DollarInput extends HTMLInputElement { ... };
            customElements.define('dollar-input', DollarInput, { extends: 'input' });
