Shadow DOM Encapsulates stylesheets

Stylesheets withing shadow DOM do not "bleed" to the application, nor leak inside another shadow DOM. When we write a custom elements that uses shadow DOM api, we enable the element to use internal stylesheet and the SLOT api.

This is styled by the global document stylesheet.

The custom element's button is not affected.



            class CustomElement extends HTMLElement {
              constructor () {
                super();
                this.attachShadow({mode: 'open'});
                this.shadowRoot... // do whatever
              }
            };
            customElements.define('custom-element', CustomElement);

Css variables can be passed through shadow DOM, very effective for theming or exposing style API for an element

Open the developer tools and see the documents style, and how it affects the custom element
